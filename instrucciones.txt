## Puedo clonar el repo de GitHub: git clone https://github.com/vviotti/Modelo-Trading-Rofex
## Antes activar el entorno virtual: .\venv\Scripts\Activate.ps1
	Si el entorno no está instalado --> python -m venv venv
## Instalar librerías: pip install -r requirements.txt

1) Realizar el backtesting en optimizar.py (python optimizer.py)
2) Aplicar los parámetros en config.py
3) Subir a GitHub
	git add .
	git commit -m "Commit"
	git push -u origin main
4) Ejecutar Bot (python live_bot.py)
